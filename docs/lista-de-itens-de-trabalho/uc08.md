## UC08 — Ler Token para Descarte
 
**Atores:** Usuário

**Objetivo:** Permitir leitura de QR Code para descarte.

**Pré-condições:**

- Usuário autenticado.
- PEV possuir QR Code válido.

**Fluxo Principal**

1. Usuário acessa funcionalidade de leitura.
2. Sistema ativa a câmera do dispositivo (FE-E1).
3. Usuário aponta a câmera para o QR Code.
4. Sistema realiza leitura do token (FA-4A).
5. Sistema valida o token recebido, incluindo verificação de validade/expiração. (RN2) (RN10) (FE-E3)
6. Sistema verifica se o token já foi utilizado anteriormente. (RN6) (FE-E4)
7. Sistema extrai do token os dados do descarte informados pelo PEV (tipo de resíduo e volume/peso).
8. Sistema verifica se o material ou tipo de resíduo descartado é aceito pelo PEV. (RN16) (FE-E5)
9. Sistema registra a operação de descarte (FE-E2).
10. Sistema converte o volume e/ou tipo de resíduo descartado em pontos e credita o saldo do usuário, conforme tabela de pontuação vigente. (RN3)
11. Sistema confirma leitura realizada.
12. Sistema verifica se o usuário atingiu critérios para obtenção de novas insígnias. (RN4)

**Fluxos Alternativos**

- **FA-4A — QR Code inválido**

    - 4A.1 Sistema não reconhece o código.
    - 4A.2 Sistema solicita nova leitura.

**Fluxos de Exceção**

- **FE-E1 — Falha ao acessar a câmera**

    - E1.1 Sistema não inicia a leitura do QR Code.
    - E1.2 Sistema informa que a permissão ou o recurso de câmera está indisponível.

- **FE-E2 — Falha ao registrar o descarte**

    - E2.1 Sistema não registra a operação de descarte.
    - E2.2 Sistema não credita pontos ao usuário. (RN3)
    - E2.3 Sistema não realiza a verificação de novas insígnias. (RN4)
    - E2.4 Sistema informa que a operação não foi concluída.
    
- **FE-E3 — Token expirado**

    - E3.1 Sistema detecta token inválido ou expirado.
    - E3.2 Sistema bloqueia a operação.

- **FE-E4 — Token já utilizado**

    - E4.1 Sistema identifica reutilização do token. (RN6)
    - E4.2 Sistema impede o registro do descarte.
    - E4.3 Sistema informa que o token já foi utilizado anteriormente.

- **FE-E5 — Material não aceito pelo PEV**

    - E5.1 Sistema identifica que o material ou tipo de resíduo informado não consta entre os aceitos pelo PEV. (RN16)
    - E5.2 Sistema impede o registro do descarte.
    - E5.3 Sistema informa ao usuário que o material não é aceito naquele ponto de coleta.
        
**Pós-condições:**

- Sucesso: descarte registrado, pontos creditados e elegibilidade de insígnias avaliada.
- Falha: nenhum descarte é registrado como válido e nenhum ponto é creditado.
