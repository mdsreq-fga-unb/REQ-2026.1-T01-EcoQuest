## UC15 — Visualizar Ranking
 
**Atores:** Usuário

**Objetivo:** Exibir ranking social com pontuação dos usuários.

**Pré-condições:** Usuário autenticado.

**Fluxo Principal**

1. Usuário acessa o ranking.
2. Sistema recupera a classificação dos usuários com base na pontuação acumulada pelos descartes realizados. (RN3)
3. Sistema exibe lista ordenada por pontuação.
4. Sistema destaca a posição do usuário no ranking.
5. Usuário visualiza o ranking.

**Fluxos Alternativos**

- **FA-2A — Usuário filtra o ranking por período ou categoria**

    - 2A.1 Usuário seleciona um filtro disponível (ex.: ranking semanal, mensal ou por região).
    - 2A.2 Sistema recalcula a classificação conforme o filtro selecionado.
    - 2A.3 Fluxo retorna ao passo 3 do Fluxo Principal.

**Fluxos de Exceção**

- **FE-E1 — Falha ao carregar ranking**

    - E1.1 Sistema não exibe classificação incompleta como definitiva.
    - E1.2 Sistema informa indisponibilidade temporária do ranking.

- **FE-E2 — Falha ao aplicar preferências de anonimato**

    - E2.1 Sistema preserva a privacidade configurada pelos usuários.
    - E2.2 Sistema informa que o ranking não pôde ser exibido no momento.

- **FE-E3 — Ranking indisponível**

    - E3.1 Sistema não consegue carregar os dados.
    - E3.2 Sistema informa indisponibilidade temporária.
        
**Pós-condições:** Ranking exibido ao usuário.
