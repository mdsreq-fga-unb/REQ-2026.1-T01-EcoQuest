## UC11 — Resgatar Recompensas
 
**Atores:** Usuário

**Objetivo:** Processar o resgate de uma recompensa do catálogo.

**Pré-condições:**

- Usuário autenticado.

**Fluxo Principal**

1. Usuário seleciona uma recompensa no catálogo.
2. Sistema exibe detalhes e custo em pontos da recompensa conforme configuração vigente, validando disponibilidade e estoque do item. (RN1) (RN13) (RN17) (FE-E3)
3. Usuário confirma o resgate.
4. Sistema verifica se o usuário possui saldo igual ou superior ao valor exigido para a recompensa selecionada. (RN5) (FA-4A)
5. Sistema gera o código ou cupom de benefício e reserva uma unidade do estoque da recompensa. (RN17) (FE-E2)
6. Sistema debita os pontos do saldo somente após a confirmação da geração do cupom. (FE-E1)
7. Sistema confirma o resgate realizado.

**Fluxos Alternativos**

- **FA-4A — Saldo insuficiente**

    - 4A.1 Sistema identifica que o usuário não atende ao requisito mínimo de pontos para o resgate. (RN5)
    - 4A.2 Sistema bloqueia a operação e informa o usuário. Nenhum cupom é gerado e nenhum ponto é debitado.

**Fluxos de Exceção**

- **FE-E1 — Falha ao debitar pontos após geração do cupom**

    - E1.1 Sistema identifica falha na etapa de débito, após o cupom já ter sido gerado e o estoque reservado.
    - E1.2 Sistema cancela o cupom gerado e libera a unidade reservada do estoque.
    - E1.3 Sistema informa que o resgate não foi concluído.

- **FE-E2 — Falha ao gerar código ou cupom**

    - E2.1 Sistema não reserva unidade do estoque da recompensa.
    - E2.2 Sistema não debita pontos do saldo do usuário.
    - E2.3 Sistema informa indisponibilidade temporária do resgate.

- **FE-E3 — Recompensa esgotada**

    - E3.1 Sistema identifica item sem estoque disponível. (RN17)
    - E3.2 Sistema informa indisponibilidade e impede a seleção do item para resgate.

**Pós-condições:**

- Sucesso: cupom gerado, estoque da recompensa atualizado e pontos debitados do saldo do usuário.
- Saldo insuficiente: nenhum cupom gerado, nenhum ponto debitado, recompensa permanece disponível no catálogo.
- Recompensa indisponível/esgotada: nenhum cupom gerado, nenhum ponto debitado.
- Falha na geração do cupom: nenhum cupom gerado, nenhum ponto debitado, estoque não é reservado.
